Build a web-based application that analyzes legal contracts using Natural Language Processing (NLP) and Machine Learning. The system should allow users to upload legal documents and automatically extract clauses, classify them, detect risk levels, answer user queries, and simplify legal language.

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
